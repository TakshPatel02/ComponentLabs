# ComponentLabs

ComponentLabs is a premium React component library built with Vite, Tailwind CSS v4, and Framer Motion. It features high-fidelity, interactive UI components designed for modern web applications.

## 📂 Project Structure

```text
ComponentLabs/
├── public/                # Static assets
├── src/
│   ├── assets/            # Project images and icons
│   ├── components/        # Reusable UI components
│   │   ├── ButtonComponents/
│   │   ├── CardComponents/
│   │   ├── FormComponents/
│   │   ├── GridComponents/
│   │   ├── HeroComponents/
│   │   ├── HomePageComponents/
│   │   ├── KeyBoardComponents/
│   │   ├── Linkcomponent/
│   │   ├── OtherComponents/
│   │   ├── TextComponents/
│   │   ├── Footer.jsx
│   │   ├── NavBar.jsx
│   │   ├── ScrollToTop.jsx
│   │   └── ThemeToggle.jsx
│   ├── context/           # React Context providers (e.g., ThemeContext)
│   ├── Pages/             # Application pages
│   │   ├── Components/    # Individual component showcase pages
│   │   ├── ComponentsPage.jsx
│   │   ├── DocsPage.jsx
│   │   ├── HomePage.jsx
│   │   └── TermsPage.jsx
│   ├── App.jsx            # Root application component
│   ├── index.css          # Global styles and Tailwind directives
│   └── main.jsx           # Application entry point
├── README.md              # Project documentation
├── package.json           # Project dependencies and scripts
├── vite.config.js         # Vite configuration
├── PUBLISHING_GUIDE.md    # Guide for publishing to npm
└── vercel.json            # Vercel deployment configuration
```

## 🚀 Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

For detailed instructions on how to publish this library to npm, please refer to the [PUBLISHING_GUIDE.md](./PUBLISHING_GUIDE.md).
